# Teensy 4.1

Teensy 4.1 是 PJRC 推出的一款基于 NXP iMXRT1062 (ARM Cortex-M7 @ 600 MHz) 的高性能开发板，相比 Teensy 4.0 增加了以太网、Micro SD、USB Host 等接口，并提供更多 I/O 与外置存储扩展焊盘。

## 主要参数

- 处理器：NXP iMXRT1062，ARM Cortex-M7 @ 600 MHz（带浮点单元）
- RAM：1024 KB（含 512 KB tightly coupled）
- Flash：8192 KB
- 数字 I/O 引脚：55（其中 35 个可输出 PWM）
- 模拟输入：18（A0-A17，10/12 位 ADC）
- 串口：8（Serial1-Serial8）+ Serial（USB）
- SPI：3
- I2C：3
- I2S：2
- 板载 LED：D13
- 工作电压：3.3V（**不兼容 5V**）
- USB：原生 USB 480 Mbit/s（USB-C），另有 USB Host 接口
- 以太网：10/100 Mbit Ethernet PHY（DP83825）
- 板载 Micro SD 卡座
- 可选 PSRAM/Flash 焊盘
- 尺寸：61.0 × 17.78 mm（2.4 × 0.7 inch）
