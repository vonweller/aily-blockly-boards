# Teensy 4.0

Teensy 4.0 是 PJRC 推出的一款基于 NXP iMXRT1062 (ARM Cortex-M7 @ 600 MHz) 的高性能开发板。

## 主要参数

- 处理器：NXP iMXRT1062，ARM Cortex-M7 @ 600 MHz（带浮点单元）
- RAM：1024 KB（含 512 KB tightly coupled）
- Flash：2048 KB
- 数字 I/O 引脚：40（其中 31 个可输出 PWM）
- 模拟输入：14（A0-A13，10/12 位 ADC）
- 串口：7（Serial1-Serial7）+ Serial（USB）
- SPI：3
- I2C：3
- I2S：2
- 板载 LED：D13
- 工作电压：3.3V（**不兼容 5V**）
- USB：原生 USB 480 Mbit/s（USB-C）
- 尺寸：35.56 × 17.78 mm（1.4 × 0.7 inch）
