# 开发板描述多语言（i18n）规范

## 概述

开发板 `package.json` 使用 `description_<lang>` 字段提供多语言描述文本。

## 支持的语言

| 语言代码 | 语言 |
|---------|------|
| `en` | English（英语） |
| `zh_cn` | 简体中文 |
| `zh_hk` | 繁體中文（香港） |
| `ja` | 日本語（日语） |
| `ko` | 한국어（韩语） |
| `de` | Deutsch（德语） |
| `fr` | Français（法语） |
| `es` | Español（西班牙语） |
| `pt` | Português（葡萄牙语） |
| `ru` | Русский（俄语） |
| `ar` | العربية（阿拉伯语） |

## 字段规范

字段以 `description_<lang>` 命名，紧接在 `description` 字段之后：

```json
{
  "name": "@aily-project/board-arduino_uno",
  "nickname": "Arduino UNO R3",
  "version": "1.8.6",
  "description": "The Arduino UNO R3 is an open-source prototyping platform...",
  "description_en": "The Arduino UNO R3 is an open-source prototyping platform...",
  "description_zh_cn": "Arduino UNO R3 是一款基于 ATmega328P 的开源原型开发平台...",
  "description_zh_hk": "Arduino UNO R3 是一款基於 ATmega328P 的開源原型開發平台...",
  "description_ja": "Arduino UNO R3は、ATmega328Pを搭載したオープンソースプロトタイピングプラットフォームで...",
  "description_ko": "Arduino UNO R3는 ATmega328P 기반의 오픈소스 프로토타이핑 플랫폼으로...",
  "description_de": "Der Arduino UNO R3 ist eine Open-Source-Prototyping-Plattform...",
  "description_fr": "L'Arduino UNO R3 est une plateforme de prototypage open source...",
  "description_es": "El Arduino UNO R3 es una plataforma de prototipado de código abierto...",
  "description_pt": "O Arduino UNO R3 é uma plataforma de prototipagem de código aberto...",
  "description_ru": "Arduino UNO R3 — платформа для прототипирования с открытым исходным кодом...",
  "description_ar": "Arduino UNO R3 هو منصة نماذج أولية مفتوحة المصدر...",
  "main": "index.js",
  "scripts": { ... },
  ...
}
```

**字段说明：**
- `description` — 原始英文描述（保持不变）
- `description_en` — 英文描述（与 `description` 相同，用于统一 i18n 调用逻辑）
- `description_<lang>` — 对应语言的翻译文本

## 翻译要求

- **技术术语保持原文**：芯片型号（如 ATmega328P、ESP32-S3）、协议名称（如 Wi-Fi、Bluetooth、SPI、I2C）、品牌名等应保持英文原文
- **数值参数保持一致**：频率、内存大小、引脚数量等数值在所有语言中保持一致
- **描述内容一致性**：各语言翻译应传达相同的技术信息，不应遗漏或添加原文中没有的内容
- **语言风格本地化**：翻译应符合目标语言的自然表达习惯
